Computer Vision Project: Pokemon Image Classification


{% load static %}

This project takes an image set of all Pokemon from generations 1-7 and classifies which generation they come from by image alone, utilizing a deep learning Keras model. The model was able to classify the correct generation with 85% accuracy. This is not an "old Pokemon" versus "New Pokemon" split, but rather classifying the Pokemon into their specific generation number, 1-7. This result shows that there are unique characteristics of the Pokemon of each generation that goes beyond the new versus old mentality many in the fandom seem to hold.

Data Clustering Project: Bank Data Clustering


{% load static %}

This project stemmed from seeing data I had not seen up to that point and learning to wrangle it. In this case, it was putting bank data into clusters. After cleaning the data, deciding what features to use emphasize, and figuring how many clusters to use, I came out with 3 distinct cluster. In taking the mean of those in each cluster, I found three groups these bankers can fit in. Those were the low engagers (people who spend less and save less), the heavy spenders (the people who spend more and have more in payments, but keep less in the bank at a time), and the savers (those who have a lot in the bank, but rarely purchase and have less payments to make)

Data Classification Project: Census Income Classification


{% load static %}

This project takes very messy census data, cleans it up, then classifies the entries by their income. There were two income brackets being classified, being less than or equal to 50K a year and more than 50K a year. This project compares multiple different methods of classification, maxing out at an accuracy of about 85%. The difference between the lowest accuracy Decision Tree and the highest accuracy Gradient Boost was about 6%.
