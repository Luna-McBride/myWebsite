Tableau Project: Chase Bank Data

{% load static %}

This project was a combination of SSIS and Tableau to gain information from deposit data Chase Bank posted on Kaggle (https://www.kaggle.com/chasebank/bank-deposits). SSIS was done first to make sure there were no problems with the data when put into a SQL Server database. This was then pushed back to an Excel file (since free Tableau does not have direct integration with SQL Server) and various graphs were then made in Tableau to gain insight on the data. If you would like to see the data and graphs, see the Github link below.
